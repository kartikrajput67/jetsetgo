# jetsetgo
# JetSetGo Backend

## Setup Instructions

1. Clone repo
2. Run: npm install
3. Create .env file:
PORT=3000

4. Setup MySQL in config/config.json
5. Run: npm run dev

## Tech Stack
- Node.js
- Express.js
- MySQL
- Sequelize

## Notes
- Uses Sequelize ORM
- Tables created using sequelize.sync()

## Features
- REST API structure
- MVC architecture
- Sequelize ORM integration
